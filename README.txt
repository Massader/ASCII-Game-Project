To run the game properly, run the source code or the jar with a console which supports ANSI escape codes.
Make sure to add the levels_dir folder in src as an arguement

This is a game project programmed in Java, made for an OOP class. It implements console text-based UI and several design patterns. In the game you play as a chosen Game of Thrones
character on a grid map and must fight enemies to progress through the levels and eliminate the Night King. Have fun!

Controls:
W - Move up
A - Move left
S - Move down
D - Move right
E - Special Attack (unique to each class)
Q - Skip turn

To attack an enemy, simply walk towards them.
